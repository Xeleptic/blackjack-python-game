This is a fully functioning single-player blackjack game coded in Python using PyCharm. 
The functions required to run both the base game (blackjack_regular) and the extended game (blackjack_extended) are in the blackjack_functions file, while the other two files simply run the selected game using the functions.
The base game contains a simple single-player experience where you play against the dealer in a best-of-three, while the extended version adds some context, difficulties, and gambling.
To run the game through PyCharm, proceed as follows:
1. Ensure that Python is installed on your machine
2. Download the repository as a zip and extract its contents
3. Open PyCharm and navigate to the extracted folder
4. Run: python blackjack_regular.py (for the base game)
   Or: python blackjack_extended.py (for the extended version)
5. Follow the instructions in the terminal to play

These instructions are written for PyCharm; however, the process should be similar for other platforms. Enjoy :)
